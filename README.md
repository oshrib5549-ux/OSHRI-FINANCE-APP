# Oshri Finance App (Hebrew RTL)

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Deploy to Vercel
1. vercel.com → Login
2. New Project → Import → upload this ZIP
3. Framework: **Vite**
4. Build Command: `npm run build`
5. Output Directory: `dist`
